# Chatbot
Chat bot UI Implementation with ractive and vue Js with Websocket implementation

Bot will communicate and update chat window based on web socket response in real time

## Tools / Technologies
- Vue.js
- SASS
- Webpack
- Websocket

## Message types
- Text Message
- options/topics
